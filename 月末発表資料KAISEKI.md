### セクション1：導入・問題提起（0:00-1:00）

**キーメッセージ**：「人が書いた文章から、書き手の心理状態を自動的に読み取れるか？」

**話す内容**：
- 言語は人間の認知・感情・社会的行動を反映する「窓」である（Pennebaker, 2011）
- LIWCは20,000以上の論文で使用されたゴールドスタンダード
- しかし日本語では根本的な限界がある → それを**AIで超える**のが本研究

**注意点**：
- 聴衆の専門は多様なので、なるべく平易な導入にする
- 「なぜこの研究が面白いか」をまず伝える

### セクション2：背景──LIWC/J-LIWCとその限界（1:00-3:00）

**キーメッセージ**：「辞書ベースのアプローチは文脈が読めない」

**話す内容**：

1. **LIWCとは**：テキスト中の単語を約100カテゴリに分類し、出現割合を算出するソフトウェア
2. **J-LIWC2015**（五十嵐・奥田・笹原, 2022）：日本語版の開発と貢献
3. **辞書ベースの限界**：

| 問題 | 具体例 |
|------|--------|
| **文脈非理解** | 「やばい」→ 辞書では常にネガティブ。「やばいくらい楽しかった」は誤分類 |
| **新語非対応** | 辞書更新が手動で数年かかる |
| **日本語固有** | 終助詞（「ね」「よ」）、敬語、主語省略の解析不能 |

4. **開発者自身も限界を認識**（Tausczik & Pennebaker, 2010）

**視覚資料（スライド案）**：
- 「やばい」の誤分類を示す具体例の図

### セクション3：KAISEKIシステムの概要（3:00-5:00）

**キーメッセージ**：「辞書＋AIのハイブリッドで文脈を読む」

**話す内容**：

1. **基本アーキテクチャ**：

```
従来型（J-LIWC）：
  テキスト → 単語分割 → 辞書照合 → カウント

新型（KAISEKI）：
  テキスト → AI（LLM）が文脈を読む → カテゴリ判定
            → 辞書型分析も並行して実施
            → 構造分析（語彙多様性・構文複雑性）
```

2. **KAISEKIの4つのレイヤー**：
   - レイヤー1：語彙分析（J-LIWC互換カテゴリ）
   - レイヤー2：文脈・意味分析（LLMによる文脈判定）← **差別化ポイント**
   - レイヤー3：構造分析（語彙多様性MTLD、構文複雑性）
   - レイヤー4：意味空間分析（テキスト間類似度）

3. **J-LIWCとの差別化ポイント**（表で端的に示す）

| J-LIWC | KAISEKI |
|--------|---------|
| 文脈理解 ✕ | ○（AI） |
| 終助詞・敬語 ✕ | ○ |
| 辞書自動更新 ✕ | ○ |
| 語彙多様性 ✕ | ○ |

### セクション4：研究目的と仮説（5:00-7:00）

**キーメッセージ**：「作ったシステムが本当に正しく動くかを三角測量で検証する」

**話す内容**：

1. **研究目的**：
   - **主目的**：KAISEKIを開発し、構成概念妥当性を検証する
   - **副目的**：J-LIWCとの比較、生理指標との関連検証

2. **構成概念妥当性とは**（図で説明）：

```
KAISEKIが「この人は不安だ」と判定 ─┐
                                      ├→ 3つが一致 ＝ 妥当性あり
質問紙でも不安スコアが高い ──────┤
                                      │
体も緊張していた（汗↑、心拍↑）──┘
```

3. **仮説**：
   - H1：KAISEKIの感情カテゴリスコアは、質問紙の感情状態と正の相関を示す
   - H2：3条件間（ポジティブ/ネガティブ/統制）で言語カテゴリスコアが理論予測通りに弁別される
   - H3：KAISEKIはJ-LIWCと同等以上の条件弁別力を示す
   - H4：感情カテゴリスコアと生理指標は相関する

### セクション5：研究デザインと手続き（7:00-10:00）

**キーメッセージ**：「五十嵐ら（2022）の手続きを踏襲し、直接比較を可能にする」

**話す内容**：

1. **実験パラダイム**：感情操作エッセイ記述パラダイム
   - 五十嵐ら（2022）がJ-LIWC2015の構成概念妥当性検証で実際に用いたデザイン
   - → J-LIWCの結果との**直接比較が可能**

2. **デザイン**：

| 項目 | 内容 |
|------|------|
| **計画** | 参加者間デザイン |
| **条件** | ポジティブ感情 / ネガティブ感情 / 統制（3条件） |
| **各条件** | 30名ずつ（計90名） |
| **割り当て** | ランダム |

3. **手続きのフロー**：

```
同意書署名 → 事前質問紙(BFS, POMS) → BIOPAC装着
    → 200文字以上エッセイ記述＋生理記録
    → 事後質問紙(POMS, 操作チェック, PANAS)
    → デブリーフィング
    所要時間：約50分
```

4. **使用する質問紙と測定**：
   - BFS（Big Five短縮版）：パーソナリティ特性
   - POMS短縮版：気分状態（事前・事後）
   - PANAS：感情状態
   - BIOPAC：EDA（皮膚電気活動）、心拍変動

5. **サンプルサイズの根拠**：
   - 効果量 f = 0.35、α = .05、1−β = .80 → G*Powerで約84名
   - 脱落考慮で90名

### セクション6：分析計画と期待される結果（10:00-12:00）

**キーメッセージ**：「KAISEKI vs J-LIWCの直接対決」

**話す内容**：

1. **分析の柱**：

| 分析 | 対象 | 手法 |
|------|------|------|
| **条件弁別** | 3条件×言語カテゴリ | 一要因ANOVA |
| **収束的妥当性** | 言語カテゴリ × 質問紙 | ピアソン相関 |
| **生理との関連** | 言語カテゴリ × EDA/HR | 相関・重回帰 |
| **システム比較** | KAISEKI vs J-LIWC | 相関係数の差の検定 |

2. **期待される結果**：
   - ネガティブ条件でネガティブ感情語↑、ポジティブ条件でポジティブ感情語↑
   - KAISEKIの方がJ-LIWCより条件弁別力が高い（文脈理解の優位性）
   - 感情カテゴリスコアと生理指標の相関が確認される

3. **結果が予測と異なった場合の解釈**：
   - KAISEKIとJ-LIWCが同等 → LLM型の優位性は限定的、辞書型の堅牢性を示唆
   - 生理との相関なし → 言語表出と生理反応の乖離（表出抑制等の可能性）

#### 余語真夫 先生（生理心理学・健康心理学）

**Q1：「BIOPACでEDAを測定するとのことだが、筆記中に手指にセンサをつけたままタイピングできるのか？ 運動アーチファクトの混入はどう対処するのか」**

**A1**：
ご指摘ありがとうございます。EDAセンサは**非利き手**の第2・第3指の腹に装着するため、筆記（タイピング）は利き手で行います。ただし、参加者のタイピングスタイルが両手使用である場合にはアーチファクトが混入する可能性があります。対処として：
1. 事前にタイピングスタイル（片手/両手）を確認し、共変量として統計的に統制する
2. 運動アーチファクトが疑われる波形区間をAcqKnowledgeで視覚的にスクリーニングし除外する
3. 必要に応じて、EDA解析ではトニックレベル（SCL：皮膚コンダクタンス水準）を主指標とし、フェイジック成分（SCR）は運動アーチファクトの影響が少ない区間に限定する

**Q2：「心拍変動（HRV）を分析するとのことだが、200文字の記述時間は何分程度を想定しているか。RMSSDやLF/HFの算出には最低5分程度の連続データが必要とされるが」**

**A2**：
重要なご指摘です。記述時間は個人差がありますが、200文字以上という下限設定の場合、短い方で3〜5分、長い方で10分程度になると想定しています。HRVの周波数領域分析（LF/HF）には確かに5分間の安定データが推奨されますので（Task Force, 1996）：
1. 時間領域指標（RMSSD、SDNN）を主指標とし、これらは比較的短い区間でも算出可能
2. 周波数領域分析は、記述時間が5分以上の参加者に限定して探索的に実施
3. ベースライン測定（3分）との差分を用いることで、個人差を統制

#### 大平英樹 先生（感情心理学・生理心理学・予測符号化）

**Q3：「KAISEKIが利用するLLMは、学習データに含まれるバイアスの影響を受ける可能性がある。LLMの判定結果の妥当性をどのように担保するのか。循環論法に陥らないか」**

**A3**：
極めて重要なご指摘です。LLMの学習バイアスの問題は認識しており、以下のように対処します：
1. **外的基準との照合**：KAISEKIの判定結果を、独立した外的基準（質問紙・生理指標）と照合することで、LLM内部のバイアスに依存しない妥当性検証を行います。これが三角測量アプローチの核心です
2. **人間評価者との一致率**：心理学の訓練を受けた評価者による手動コーディングとKAISEKIの判定の一致率（Cohen's κ）を算出し、AIの判定が人間の判断と整合するかを確認します
3. **既知の正解がある課題での事前検証**：感情操作による3条件設計は「正解ラベル」が明確であるため（ポジティブ条件→ポジティブ語が多いはず）、LLMのバイアスが検出可能です
4. 循環論法の回避：KAISEKIの出力は**LIWCと同等のカテゴリスコア**という解釈可能な形式であり、ブラックボックスな分類ではありません

**Q4：「感情を『ポジティブ/ネガティブ』という次元で扱っているが、これは構成主義的感情理論（Barrett, 2006）からすると過度に単純化されていないか。感情のカテゴリ構造についてどう考えるか」**

**A4**：
ご指摘の通り、構成主義的感情理論の観点からは、感情を離散カテゴリや快-不快の二次元に還元することへの批判があります。本研究では：
1. **実用的理由**：J-LIWC2015との直接比較を目的としているため、同一のカテゴリ構造（posemo/negemoの二値）を採用しています
2. **理論的発展の余地**：KAISEKIのレイヤー2（文脈・意味分析）では、LLMが感情の粒度をより細かく判定できるため、将来的には離散感情カテゴリ（怒り・悲しみ・恐怖等）や感情の混合（mixed emotions）の検出も技術的に可能です
3. **本研究の位置づけ**：まず既存のLIWCカテゴリとの互換性で妥当性を確立し、その上でKAISEKI独自のより精緻な感情分析の妥当性を検証するという段階的アプローチをとっています

#### 佐藤健二 先生（臨床心理学・感情研究）

**Q5：「BFS（Big Five短縮版）とPOMSの両方を使う理由は何か。特性と状態の区別はどう分析に反映されるのか」**

**A5**：
BFSは**特性レベル**のパーソナリティを測定し、POMSは**状態レベル**の気分を測定するものです。これらを両方使用する理由は：
1. **特性-言語対応**：Pennebaker & King（1999）が示したように、Big Fiveの特性と言語スタイルの間には安定的な相関があります（例：神経症傾向→ネガティブ感情語）。これはKAISEKIの**収束的妥当性**の検証に使用します
2. **状態-言語対応**：POMS（特に事前事後の差分）は、感情操作（3条件）による**状態変化**が言語に反映されるかを検証するために使用します。操作チェックとしての機能も兼ねています
3. **分析上の区別**：特性変数（BFS）は共変量として投入し、状態変数（POMS変化量、PANAS）は条件の主効果の分析に使用するという形で区別します

**Q6：「ネガティブ感情条件で参加者が不快な記述を求められるが、その後のフォローアップは十分か。特に臨床的なリスクをもつ参加者への配慮は」**

**A6**：
1. **除外基準**を設け、精神疾患治療中の者、過去6か月以内に重篤なトラウマ体験があった者は参加を除外します
2. デブリーフィングでは、実験の趣旨を説明し、不快な感情が残っていないかを確認します
3. 大学の**心理相談室の連絡先**を記載したカードを配布します
4. 先行研究（五十嵐ら, 2022; 及川・及川, 2012）でも同様の感情操作課題が使用されており、一時的な感情喚起にとどまることが確認されています

#### 中村靖子 先生（社会心理学）

**Q7：「KAISEKIの独自性として『終助詞』や『敬語』の分析を挙げているが、本研究の実験パラダイム（エッセイ記述）でそれらの特徴が十分に出現するか。対人場面でないと敬語は使われないのでは」**

**A7**：
的確なご指摘です。確かにエッセイ記述という課題設定では、敬語や終助詞の使用は限定的になる可能性があります。
1. **本研究での位置づけ**：本研究は主にKAISEKIの**基本的な感情カテゴリの妥当性**を検証する第一段階です。終助詞・敬語分析はKAISEKIの**技術的能力**として紹介しますが、それらの妥当性検証は対人場面を含む将来研究の課題と明確に位置づけます
2. **ただし出現可能性あり**：エッセイでも「〜ですね」「〜と思いました」のような丁寧体と常体の混在が見られることがあり、それ自体が書き手の心理状態（フォーマリティの意識）を反映する指標になりえます
3. **将来研究**として、SNS投稿やLINEメッセージなどの日常的対人コミュニケーションデータへの適用を検討しています

**Q8：「200文字以上という文字数条件だが、条件間で文字数に差が出た場合、言語カテゴリの出現頻度に交絡する可能性はないか」**

**A8**：
ご指摘の通り、文字数は交絡変数になりえます。対処として：
1. KAISEKIおよびJ-LIWCの出力は、**相対頻度**（総語数に対する各カテゴリの割合）で算出するため、絶対的な文字数の差異の影響は軽減されます
2. さらに、**総文字数を共変量として投入した共分散分析（ANCOVA）**を実施し、文字数の影響を統計的に統制します
3. 条件間で文字数の分布を事前に確認し、極端な差がある場合は報告します

#### 上野大介 先生（方法論）

**Q9：「効果量 f = 0.35 を想定しているが、その根拠は何か。五十嵐ら（2022）の実際の効果量はいくつだったのか」**

**A9**：
五十嵐ら（2022）のStep 8（構成概念妥当性検証）の結果では：
1. ネガティブ感情条件とポジティブ感情条件の間のネガティブ感情語の差：中〜大程度の効果量が報告されています
2. f = 0.35は、Cohenの基準でいう「中〜大の中間」に相当し、感情操作という明確な操作があることから妥当な想定と考えています
3. ただし、**保守的な見積もり**として f = 0.25（中程度）で再計算した場合、必要サンプルサイズは約159名となり、本研究の90名では検出力が不足する可能性があります。この点は**限界として明記**し、有意でなかった場合には検出力不足の可能性を議論します

**Q10：「参加者間デザインで3条件×30名としているが、参加者内デザインにしなかった理由は何か。検出力の観点からは参加者内の方が有利では」**

**A10**：
ご指摘の通り、参加者内デザインの方が検出力は高くなります。参加者間デザインを採用した理由は：
1. **五十嵐ら（2022）の手続きとの直接比較**：J-LIWC2015の構成概念妥当性検証（Step 8）が参加者間デザインで実施されているため、同一手法での比較を優先しました
2. **キャリーオーバー効果の回避**：感情操作課題は前の条件の感情的影響が次の条件に持ち越される可能性が高く、参加者内デザインでは操作の独立性が担保しにくいです
3. **参加者の負担軽減**：3条件すべてを1人が実施すると所要時間が大幅に延び、疲労効果が結果に影響する可能性があります

### 学生からの想定質問

**Q13：「LLMを使うということは、毎回の判定結果が変わる可能性があるのでは。再現性は大丈夫か」（大学院生・同志社）**

**A13**：
重要なご指摘です。LLMの出力は確率的であるため、温度パラメータ（temperature）を0に設定して決定論的出力にします。さらに、同一テキストに対する複数回の判定の一致率（テスト-リテスト信頼性）を事前に検証し、結果として報告します。

**Q14：「KAISEKIの解析結果はLLMのバージョンに依存するが、将来LLMが更新された場合、過去の研究結果との比較はどうなるか」（修士・京都女子大）**

**A14**：
この問題は従来のLIWCの辞書更新問題（LIWC2015→LIWC-22で結果が変わる）と本質的には同じです。対処として：
1. 使用したLLMのバージョンと設定を論文に明記する
2. KAISEKIのレイヤー1（辞書型分析）は決定論的であり、LLM非依存
3. 標準テストセット（ベンチマーク・テキスト集）を作成し、バージョン間の差異を定量化可能にする

**Q15：「90名のサンプルサイズで、KAISEKIとJ-LIWCの差を検出するのに十分か」（修士・徳島大）**

**A15**：
相関係数の差の検定（Steiger, 1980）の検出力は、差のサイズとN、そして2つの相関の従属性に依存します。先行研究から予想される相関差（Δr ≈ 0.10〜0.20）に対しては、N=90では検出力が十分でない可能性があります。この点は限界として認識しており、効果量と信頼区間を報告することで、有意/非有意の二分法に陥らない解釈を行います。

## 発表のポイントと注意事項

### 全体的な発表戦略

1. **ストーリー重視**：技術の詳細より「なぜこの研究が必要か」「何が面白いか」を中心に
2. **具体例で示す**：「やばい」の誤分類例は必ず使う（全員がすぐに理解できる）
3. **先行研究への敬意**：J-LIWC（五十嵐ら, 2022）の貢献を十分に認めた上で限界を指摘する
4. **三角測量の図**：構成概念妥当性の考え方は図で示す（言語×質問紙×生理）

### 避けるべきこと

- J-LIWCを「ダメなシステム」として批判しすぎない
- 技術的なLLMの詳細に時間を使いすぎない
- 90名で全てが解決するかのような楽観的な主張
- 未完成のシステムについて過度に断定的な性能主張

### 強調すべきこと

- 五十嵐ら（2022）と**同一手法**での比較は強い証拠になる
- 質問紙だけでなく**生理指標**も併用する独自性
