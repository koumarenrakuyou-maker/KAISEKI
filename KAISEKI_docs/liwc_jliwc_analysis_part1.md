# 1. LIWC-22 分析フローの詳細手順

## Step 1: 研究設計・仮説設定
1. 研究課題の明確化（LIWCで検証可能な心理言語学的仮説の策定）
2. LIWC変数への理論的構成概念のマッピング（例：「疎外感」→社会カテゴリ）
3. 分析対象カテゴリの事前選定（117カテゴリから選択）

## Step 2: テキストデータ収集
1. テキストソースの選定（インタビュー、SNS投稿、ナラティブ、会話等）
2. データ量の確認（信頼性のある推論には4000〜5000語、もしくは200ツイート以上が必要）
3. データのフォーマット確認（txt, docx, pdf, csv, xlsx対応）

## Step 3: テキスト前処理
1. **文字起こし**：音声データの場合、テキスト化（非流暢性の処理方針決定）
2. **スペルチェック**：すべてのスペルミスを修正（辞書マッチングの精度に直結）
3. **標準化**：略語の展開、標準スペリングの統一
4. **ノイズ除去**：転記コメント、メタ情報、非テキスト要素の除去
5. **ファイル形式変換**：対応形式への変換
6. **データ構造化**：スプレッドシートの場合、分析対象列の指定

## Step 4: LIWC分析の実行
1. **データインポート**：テキストファイルのロード
2. **辞書選択**：LIWC-22辞書（12,000語以上、117カテゴリ）の選択
3. **カテゴリ設定**：分析対象カテゴリの有効化/無効化
4. **セグメント設定**：テキスト分割方法の決定
5. **分析実行**：「Run LIWC」機能の実行
6. **出力生成**：カテゴリごとの頻度（総語数に対する割合）の算出

## Step 5: 追加分析モジュール（LIWC-22固有）
1. **Narrative Arc分析**：テキストをセグメント分割し、Staging/Plot Progression/Cognitive Tensionを分析
2. **Language Style Matching (LSM)**：テキスト間の言語スタイル類似度の算出
3. **Meaning Extraction Method (MEM)**：主成分分析によるトピックモデリング
4. **要約変数の算出**：Analytical Thinking, Clout, Authenticity, Emotional Tone

## Step 6: 出力解釈・統計分析
1. **変数理解**：出力値の意味（大半がテキスト全体に対する割合%）
2. **記述的分析**：コーパス間・グループ間の差異推定
3. **予測モデリング**：LIWCスコアを特徴量とした回帰/分類モデル構築
4. **仮説検定**：t検定等による群間比較
5. **制限事項の考慮**：文脈非依存性、ドメイン特異性等

# 2. J-LIWC2015 分析フローの詳細手順

## Step 1: 環境構築
1. **LIWCライセンス取得**：LIWC2015またはLIWC-22のシリアル番号が必要
2. **R環境構築**：R 4.2.0以降のインストール（UTF-8デフォルト対応）
3. **jliwcパッケージのインストール**：`remotes::install_github()`経由でGitHubからインストール
4. **辞書ファイルのインストール**：IPAdic、ユーザー辞書、J-LIWC2015辞書の3種類

## Step 2: テキストデータ準備
1. **日本語テキスト収集**：対象テキストの収集
2. **エンコーディング確認**：UTF-8への統一
3. **ノイズ除去**：誤字、単独ひらがな文字、非辞書語の除去
4. **テキスト正規化**：全角/半角統一、表記ゆれの修正

## Step 3: 形態素解析（日本語特有の前処理）
1. **MeCab形態素解析の実行**：IPAdicを使用した単語分割
2. **ユーザー辞書適用**：複合語の不適切な分割を防止
3. **品詞タグ付け**：各形態素への品詞情報付与
4. **形態素解析結果の確認**：分割精度の検証

## Step 4: J-LIWC辞書マッチング
1. **辞書ロード**：J-LIWC2015辞書（11,609語、69カテゴリ）の読み込み
2. **単語照合**：形態素解析結果とJ-LIWC辞書のマッチング
3. **カテゴリ割当て**：一致した単語への心理カテゴリの付与
4. **カバレッジ率の確認**：辞書カバレッジの計算（約60%〜）

## Step 5: 分析実行
1. **Rスクリプト実行**：`jliwc`パッケージの分析関数呼び出し
2. **カテゴリ頻度算出**：各心理カテゴリの出現割合計算
3. **MeCabカテゴリ出力**：品詞分析用のMeCabカテゴリデータ出力

## Step 6: 結果解釈
1. **J-LIWCカテゴリの解釈**：69カテゴリのスコア分析
2. **英語版との比較時の注意**：冠詞・前置詞・比較級・時制カテゴリが除外されている点
3. **品詞分析**：MeCabカテゴリの活用推奨
4. **統計分析**：適切な統計手法の適用

## J-LIWCにおける日本語特有の課題
- 日本語にはスペースによる単語区切りがないため、形態素解析が必須
- 冠詞（a, the）が存在しないためカテゴリ除外
- 前置詞が存在しないためカテゴリ除外
- 比較級は副詞・助動詞として表現されるため該当カテゴリ除外
- 時制は助動詞で表現されるため時間指向カテゴリの扱いが異なる
- 敬語体系（丁寧語・尊敬語・謙譲語）への対応が不十分

# 3. LIWC-22 主要カテゴリ構造

## 要約変数（Summary Variables）
- Analytical Thinking（分析的思考）
- Clout（社会的影響力）
- Authenticity（真正性）
- Emotional Tone（感情トーン）

## 言語次元（Linguistic Dimensions）
- 語数、一文あたりの語数、辞書語の割合
- 機能語（代名詞、冠詞、前置詞、助動詞、接続詞、否定語）
- 人称代名詞（1人称単数/複数、2人称、3人称単数/複数）

## 心理プロセス（Psychological Processes）
- **感情**：肯定感情、否定感情、不安、怒り、悲しみ
- **認知プロセス**：洞察、因果、不確実性、確信
- **社会プロセス**：家族、友人、男性参照、女性参照
- **動機**：達成、権力、報酬、リスク

## 拡張カテゴリ
- 文化、ライフスタイル、身体、個人的状態、知覚、時間指向、会話変数

# 4. J-LIWC2015 カテゴリ構成（69カテゴリ）

LIWC2015の構造を基盤としつつ、日本語に不適合なカテゴリを除外：
- **除外カテゴリ**：冠詞、前置詞、比較級、時間指向（過去/現在/未来）
- **含まれるカテゴリ**：感情、認知、社会、動機、身体、知覚等の心理カテゴリ
- **辞書規模**：11,609語（LIWC-22の12,000語超に比べやや少ない）
