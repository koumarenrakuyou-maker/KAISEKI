<h1>サンプルサイズ（実験参加者数）の根拠と解説</h1>

本文書は、KAISEKI構成概念妥当性検証実験（候補1：筆記開示パラダイム）に必要な参加者数について、統計的根拠を整理したものである。

<h2>中学生にもわかる説明：なぜ「人数」が大事なのか</h2>

<h3>たとえ話：コインが「イカサマ」かどうか見破る実験</h3>

ここに1枚のコインがある。普通のコインなら表と裏は50%ずつ出るはず。でも「このコインは55%の確率で表が出るイカサマコインだ」と誰かが言っている。これを確かめたい。

- **5回投げた場合**：3回表が出ても「まあ普通だな」としか思えない。5回程度では「55%なのか50%なのか」の違いは見分けられない。
- **100回投げた場合**：55回表が出たら「ちょっと多いかも…」と思い始めるが、まだ偶然かもしれない。
- **1,000回投げた場合**：550回表が出たら「これは偶然じゃない、やっぱりイカサマだ」と自信を持って言える。

**ポイント：見つけたい差が小さいほど、たくさんのデータ（＝たくさんの参加者）が必要になる。**

<h3>今回の研究に置き換えると</h3>

KAISEKIが「この人はネガティブ感情が高い」と判定したとき、質問紙でも「ネガティブ気分が高い」と回答しているか？　この**一致度**（相関係数r）を調べたい。

- 一致度が**大きい（r = .50）**場合 → 少人数（30人くらい）でも見つけられる
- 一致度が**中くらい（r = .30）**場合 → 85人くらい必要
- 一致度が**小さい（r = .20）**場合 → 200人近く必要

先行研究（Pennebaker & King, 1999）を見ると、性格と言語の一致度は**r = .10〜.25**がほとんど。つまり「小さい〜中くらいの一致度」を見つけなければならないので、それなりの人数が必要になる。

<h3>「検出力」ってなに？</h3>

検出力（1−β）とは、**本当に効果があるときに、それを「ある」と正しく見つけられる確率**のこと。

```
┌────────────────────────────────────────────┐
│          検出力のイメージ                    │
│                                             │
│  本当はイカサマコインなのに…               │
│                                             │
│  検出力 50% → 2回に1回しか見破れない 😰    │
│  検出力 80% → 5回に4回は見破れる 😊        │
│  検出力 95% → ほぼ確実に見破れる 🎯        │
│                                             │
│  学術研究では「80%以上」が最低基準          │
│  （Cohen, 1988; Faul et al., 2007）         │
└────────────────────────────────────────────┘
```

人数が足りないと検出力が下がる → 本当は効果があるのに「ない」と結論してしまう（**見逃しエラー**）。これは研究として非常にもったいない。

<h2>検出力分析の計算方法</h2>

<h3>使用する公式（相関分析の場合）</h3>

Fisherのz変換に基づく検出力分析（Faul, Erdfelder, Lang, & Buchner, 2007; Cohen, 1988）：

```
n = ((z_α/2 + z_β) / z_r)² + 3

ここで：
  z_α/2 = 1.96  （有意水準α = .05 両側検定の場合）
  z_β   = 0.842 （検出力 1−β = .80 の場合）
  z_r   = 0.5 × ln((1+r)/(1-r))  （Fisherのz変換）
```

> Cohen, J. (1988). *Statistical power analysis for the behavioral sciences* (2nd ed.). Lawrence Erlbaum Associates.
> https://doi.org/10.4324/9780203771587
>
> Faul, F., Erdfelder, E., Lang, A.-G., & Buchner, A. (2007). G*Power 3: A flexible statistical power analysis program for the social, behavioral, and biomedical sciences. *Behavior Research Methods*, 39(2), 175–191.
> https://doi.org/10.3758/BF03193146

<h2>分析ごとの必要人数</h2>

<h3>分析1：相関分析（主要分析）── KAISEKIスコア × Big Five</h3>

これが**ボトルネック（最も人数を必要とする分析）**である。

| 仮定するr | z_r | 計算過程 | **必要n** | 先行研究での該当 |
|-----------|-----|---------|-----------|----------------|
| r = .50（大） | 0.549 | ((1.96+0.842)/0.549)²+3 = 29 | **29** | ほぼ報告なし |
| r = .40（中〜大） | 0.424 | ((1.96+0.842)/0.424)²+3 = 47 | **47** | 一部の強い対応のみ |
| r = .30（中） | 0.310 | ((1.96+0.842)/0.310)²+3 = 85 | **85** | 楽観的見積もり |
| r = .25（小〜中） | 0.255 | ((1.96+0.842)/0.255)²+3 = 123 | **123** | P&K(1999)の中央値付近 |
| r = .20（小） | 0.203 | ((1.96+0.842)/0.203)²+3 = 194 | **194** | P&K(1999)の多くの結果 |
| r = .15（極小） | 0.151 | ((1.96+0.842)/0.151)²+3 = 347 | **347** | 最も弱い対応 |

**先行研究の実際の効果量**（Pennebaker & King, 1999より）：

| Big Five因子 | 言語カテゴリ | 報告されたr |
|-------------|------------|------------|
| 神経症傾向 | ネガティブ感情語 | r = .19 |
| 外向性 | 社会的語彙 | r = .17 |
| 外向性 | ポジティブ感情語 | r = .13 |
| 協調性 | ポジティブ感情語 | r = .12 |
| 開放性 | 長い単語（6文字以上） | r = .20 |

→ 多くの相関が**r = .12〜.20の範囲**にあることがわかる。

<h3>分析2：対応のあるt検定 ── 感情筆記 vs. 統制筆記</h3>

公式：**n = ((z<sub>α/2</sub> + z<sub>β</sub>) / d)²**

| 仮定するd | 必要n | 備考 |
|-----------|-------|------|
| d = 0.80（大） | **15** | 感情語の条件間差が非常に大きい場合 |
| d = 0.50（中） | **34** | 一般的な条件間差 |
| d = 0.30（小〜中） | **90** | 微細な言語カテゴリの差 |

→ この分析はボトルネックにならない（少人数で検出可能）。

<h3>分析3：依存相関の差の検定 ── KAISEKI vs. J-LIWC比較</h3>

Steiger（1980）のz検定。2つの相関係数の**差**を検出するには非常に大きなnが必要。

| r_KAISEKI | r_JLIWC | 差 | 必要n |
|-----------|---------|-----|-------|
| .35 | .20 | .15 | **約350** |
| .30 | .20 | .10 | **約800** |
| .30 | .25 | .05 | **約3,000以上** |

→ **卒論レベルでは統計的検定として実施困難**。記述的比較（「KAISEKIの方が相関係数が大きい傾向にある」）として報告するのが現実的。

> Steiger, J. H. (1980). Tests for comparing elements of a correlation matrix. *Psychological Bulletin*, 87(2), 245–251.
> https://doi.org/10.1037/0033-2909.87.2.245

<h3>分析4：重回帰分析 ── 生理指標 → 言語スコア</h3>

Green（1991）の経験則：**n ≥ 50 + 8k**（k = 予測変数の数）

| 予測変数の数 | 必要n |
|-------------|-------|
| k = 2（EDA, HR） | **66** |
| k = 3（EDA, HR, HRV） | **74** |
| k = 5（EDA, HR, HRV, 交互作用項等） | **90** |

> Green, S. B. (1991). How many subjects does it take to do a regression analysis? *Multivariate Behavioral Research*, 26(3), 499–510.
> https://doi.org/10.1207/s15327906mbr2603_7

<h2>理想から妥協まで：5段階のサンプルサイズ方針</h2>

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│  ★★★★★ 理想（n = 200）                               │
│   r = .20 を80%の検出力で検出可能                       │
│   先行研究の大半の効果量をカバーできる                   │
│   全分析が十分な検出力を持つ                             │
│   → 大規模研究。卒論では非現実的な場合が多い            │
│                                                          │
│  ★★★★☆ 推奨（n = 135）                               │
│   r = .25 を80%の検出力で検出可能（脱落10%込み）        │
│   先行研究の主要な効果量の多くをカバー                   │
│   → ゼミ全体で協力すれば実現可能な水準                  │
│                                                          │
│  ★★★☆☆ 現実的（n = 85〜100）                         │
│   r = .30 を80%の検出力で検出可能                       │
│   先行研究の強めの効果のみ検出可能                       │
│   → 多くの卒論研究で採用される水準                      │
│                                                          │
│  ★★☆☆☆ 最低限（n = 60）                              │
│   r = .30 の検出力が約65%に低下                         │
│   r = .35 でようやく80%の検出力                         │
│   → 限界を論文に明記すれば許容される水準                │
│                                                          │
│  ★☆☆☆☆ 危険域（n < 50）                              │
│   主要分析の検出力が50%未満                              │
│   有意な結果が出にくく、研究の意義が薄れる              │
│   → 推奨しない                                          │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

<h3>中学生向けのまとめ：「何人集めればいいの？」</h3>

```
🎯 見つけたい「一致度」が大きい → 少人数でOK
🎯 見つけたい「一致度」が小さい → たくさん必要

先行研究を見ると、性格と言葉の一致度は「小さめ」。
だから本当は200人くらい欲しい。

でも卒論で200人は大変なので…
「85人集めて、見つけられなかった小さい一致度は
 ʼ人数が足りなかったかもしれないʼ と正直に書く」
というのが現実的な落とし所。
```

<h2>本研究での推奨記述</h2>

<h3>パターンA：n = 80〜85で実施する場合（現実的妥協案）</h3>

> 本研究の主要分析（KAISEKIカテゴリスコアとBig Five各因子の相関分析）における効果量は、先行研究（Pennebaker & King, 1999）に基づきr = .30（中程度）を仮定する。有意水準α = .05（両側検定）、検出力1−β = .80の条件で、Fisherのz変換に基づく検出力分析（G*Power 3; Faul et al., 2007）により、必要サンプルサイズはn = 85と算出された。脱落を考慮しn = 80を目標とする。なお、先行研究で報告された効果量の一部（r = .12〜.20）については検出力が不足する可能性があり、この点を本研究の限界として議論する。

<h3>パターンB：n = 120〜135で実施する場合（推奨案）</h3>

> 本研究の主要分析における効果量は、Pennebaker & King（1999）が報告した言語カテゴリとBig Five因子の相関係数（r = .12〜.25、中央値r ≈ .20）に基づき、r = .25を仮定する。有意水準α = .05（両側検定）、検出力1−β = .80の条件で、G*Power 3（Faul et al., 2007）により、必要サンプルサイズはn = 123と算出された。脱落約10%を見込み、n = 135を目標として募集する。
