## 2. 研究目的・意義

### 研究目的

#### 主要な目的

LIWC（Linguistic Inquiry and Word Count）ならびに、J-LIWC以上かつ、大規模言語モデルをベースとした、**文脈理解**や**日本語固有の言語現象**に対応した新しい言語解析システム（以下、仮名である**KAISEKI**とする）を開発し、**構成概念妥当性**を検証すること。

#### 副次的な目的

1. KAISEKIの各言語カテゴリスコアと、確立された質問紙（Big Five、POMS）による心理特性指標との妥当性を検証する。
2. 筆記中の生理指標（皮膚電気活動・心拍変動）とKAISEKIの感情カテゴリスコアとの妥当性を検証する。
3. 同一テキストに対してKAISEKIとJ-LIWC2015の両方で解析を行い、心理特性との相関強度を比較することで、KAISEKIの文脈理解に基づく優位性を検証する。

### 背景事情

LIWCは心理学における言語解析のゴールドスタンダードとして学術論文で使用されてきた。日本語版LIWCであるJ-LIWC2015は五十嵐・奥田・笹原（2022）により開発されたが、翻訳版であり辞書ベースの単語カウント法である。そのため、以下の不完全な点がある：

- **文脈依存的な意味変化の処理が不可能**（例：「やばい」のポジティブ/ネガティブ使用）
- **日本語固有の言語現象への対応が不十分**（終助詞、敬語体系、主語省略）
- **辞書の更新に時間と手間がかかる**

本研究は、近年のLLMなどの技術的発展を活用して上記の課題を解決するシステムを開発し、その妥当性を科学的に検証するものである。

### 社会的意義

1. 英語圏中心だった言語解析研究を日本語で発展させる基盤を提供する。
2. 他分野（臨床心理、教育、社会心理学等）への応用力が高い。

## 3. 研究計画

### 実施手順

感情状態の操作による構成概念妥当性の検証実験である（五十嵐・笹原, 2018の手続きに基づく）。

- **デザイン**：参加者間デザイン
- **条件**：ポジティブ感情条件、ネガティブ感情条件、統制条件の**3条件**
- **割り当て**：いずれか1条件にランダムに割り当て
- **各条件の参加者数**：30名

#### 手続きの流れ

```
1. 同意書への署名と実験説明
   ↓
2. 事前質問紙
   （属性情報、BFS、POMS短縮版）
   ↓
3. BIOPACセンサ装着・ベースライン測定
   ↓
4. 群ごとに200文字以上をPC上で記述
   ＋ 同時に生理反応を記録
   ↓
5. 事後質問紙
   （事後POMS短縮版、操作チェック〔快不快〕、インパクト、PANAS）
   ↓
6. デブリーフィング
```

### 分析方法

| 分析 | 内容 |
|------|------|
| **質問紙** | 一要因分散分析（3条件間比較） |
| **言語カテゴリスコア** | KAISEKI・J-LIWC双方について一要因分散分析 |
| **質問紙×言語** | KAISEKI・J-LIWC双方の各カテゴリスコアと質問紙指標のピアソン相関分析 |
| **生理×言語** | 筆記中の生理指標とKAISEKI・J-LIWC双方について相関・重回帰分析 |
| **システム比較** | KAISEKI・J-LIWCの分析結果について相関係数の差の検定比較 |

> 基本的に、五十嵐・笹原（2018）が構成概念妥当性の検証の際に用いた分析手法を踏襲している。KAISEKI・J-LIWCを比較し、KAISEKIが同等以上に条件を弁別できる場合、優位性の証拠とする。

## 4. 対象者（調査参加者）について

### 対象者数とその設定根拠

- **対象者総数**：**90名**

#### 設定根拠

本研究における主要な分析は、3条件（ポジティブ、ネガティブ、統制）の一要因分散分析である。五十嵐ら (2022) の手続きに準じ、各種心理効果等の効果量 f = 0.35（中〜大程度）、有意水準 α = 0.05、検出力 1 − β = 0.80 で G*Power 3 (Faul et al., 2007) により検定力分析を行った場合、要請されるサンプルサイズは約84名となる。実験途中の脱落や文字数不足による除外を考慮し全体で90名を目標とする。

### 対象者の属性

| 項目 | 内容 |
|------|------|
| **年齢層** | 18歳〜30歳 |
| **性別（取得方法）** | 出生時の性別（自由記述方式） |
| **対象** | ☑ 本学に属する大学生・大学院生（未成年者は含まれない） |
| **選択基準あり** | ☑ |
| **研究開始後の除外条件あり** | ☑ |

### 選定方針

#### 選択基準

1. 18歳以上の大学生または大学院生
2. 日本語母語話者
3. 研究内容の説明を理解し、同意書に署名した者

#### 選択基準の理由

日本語の言語解析システムであるため。

#### 除外基準

| 除外対象 | 理由 |
|----------|------|
| 現在、精神疾患の治療を受けている者 | ネガティブ条件の記述が症状悪化を招く恐れがあるため |
| 過去6か月以内に重篤なトラウマ体験があった者 | フラッシュバックを誘発する恐れがあるため |

#### 研究開始後の除外条件

- 記述が200文字未満である場合
- 記述内容が無意味・全くのナンセンスであった場合
- テーマに明らかに沿っていない場合
- 生理データに不具合があった場合
- 質問紙の回答にランダム回答パターンが見られた場合

### 募集方法

大学の授業において口頭募集及び募集チラシの配布

### 留意事項

- ☑ 研究への参加の有無や研究途中での離脱が、成績評価や単位修得に影響しない
- ☑ 研究への参加の同意は強制ではなく任意である
- ☑ 研究参加中のいかなる時でも不利益な扱いを受けることなく自由に中途離脱できる
- ☑ 研究への参加同意書は、十分な説明後、適切な時期に提出いただく
